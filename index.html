VIM

Vim is a wonderful text editor, once you get the hang of it. Extensible by plugins, and highly configurable. Everything below is installed using Vundle. See the Vimrc for a list of all plugins I use.

Notable Plugins

vim-airline

Nice status/tabline for Vim. Lots of themes available.

Github Configuration
nord-vim

The preferred theme. Consistent with the terminal theme.

Github Configuration
vim-sensible

Sane defaults everyone can agree on.

Github
Syntastic

Syntax checking for Vim. Kind of mandatory if you want to edit anything beyond textfiles.

Github Configuration
Tabular

Makes aligning text much easier.

Github

Finally: Tab autocompletions that just work. A must have.

Github
vim-grammarous

Helps you suck less at grammar. Ties in nicely with spellcheck.
